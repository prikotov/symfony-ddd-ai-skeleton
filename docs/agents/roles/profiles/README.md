# Role profile cards

Эту директорию ведёт [`Эйчар Юнг`](../team/hr_jung.ru.md).

Profile card — это рабочая карточка роли AI-агента. Она фиксирует не только текущий prompt, но и историю кадровых решений: зачем роль появилась, какие поведенческие паттерны ей заданы, какие риски обнаружены и как изменения роли повлияли на команду.

Правила:

* одна карточка на одну роль;
* имя файла совпадает с `agent` из front matter роли: `{agent}.md`;
* любые существенные изменения role file сопровождаются записью в карточке;
* гипотезы и наблюдения отделяются от фактов;
* карточка не заменяет role file, а объясняет историю и мотивацию изменений.

Шаблон: [`PROFILE_TEMPLATE.md`](PROFILE_TEMPLATE.md).
